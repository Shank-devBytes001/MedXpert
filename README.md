# $${\color{skyblue}MedXpert}$$ - Digitizing the healthcare

A modern healthcare digitization platform built to streamline paperless prescription management and medical record-keeping in India in line with the Govt of India's UHID initiative.
Built with dwani AI's advanced API to support regional languages like Kannada, prescription translation and voice based health record access.

## Getting Started


```bash
# Clone the repository
git clone https://github.com/purushothamhd/MedXpert.git

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/         # Reusable UI components
├── pages/             # Page components
├── utils/             # Utility functions
│   ├── db.ts         # Database operations
│   ├── pdf.ts        # PDF generation
│   └── auth.ts       # Authentication logic
├── data/             # Static data
└── types/            # TypeScript type definitions
```



### Key Features

1. **Multi-User System**
   - Doctor Dashboard
   - Patient Portal
   - Medical Store Interface

2. **Digital Prescription Management**
   - Handwriting Recognition for Medicine Names
   - Voice Input Support
   - PDF Generation
   - Digital Signature Support

3. **Patient Management**
   - UHID (Unique Health ID) System
   - Medical History Tracking
   - Record Management

4. **Security & Privacy**
   - Client-side Data Processing
   - Secure Document Generation
   - Privacy-focused Architecture
